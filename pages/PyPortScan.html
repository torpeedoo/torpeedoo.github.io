PyPortScan

PyPortScan is a Python based port scanning tool designed to identify open ports on a target system. It uses the Scapy and Socket libraries to send TCP SYN packets to specified ports and analyze the responses to determine their status. The tool is capable of scanning a range of ports, making it useful for network security assessments and vulnerability analysis.

Features

  • Scans a single, list, or range of ports on a target IP address.
  • Identifies open and closed ports.
  • Simple host discovery on local network.
  • Provides a simple command-line interface for ease of use and automation.

Usage

python main.py <scan_function> <target_ip> <port(s)>

Replace <scan_function> with one of the following options:

  • single: Scan a single port.
  • list: Scan a list of ports.
  • range: Scan a range of ports.
  • local: Simple host discovery on local network.

Example scan

python main.py single scanme.nmap.org 80

This command scans port 80 on the target IP address scanme.nmap.org and returns: 


   Starting scan...
   Port 80 is open on 45.33.32.156
   Total scan time: 0.12 seconds

Code Implementation

The scanner logic is implemented in a Scanner class within functions.py. It handles different scanning modes (single, list, range, and local discovery) and provides both standard and stealth scanning methods.


class Scanner:
    def __init__(self, ip, timeout=0.1):
        self.timeout = timeout
        self.ip = self.resolve_ip(ip)

    def scan_port(self, port):
        with socket.socket(socket.AF_INET, socket.SOCK_STREAM) as s:
            s.settimeout(self.timeout)
            result = s.connect_ex((self.ip, port))
            return result == 0

For stealth scanning, the tool uses scapy to send crafted SYN packets without completing the TCP handshake, reducing the likelihood of detection by intrusion detection systems (IDS).


def stealth_scan_port(self, port):
    syn_packet = IP(dst=self.ip)/TCP(dport=port, flags='S')
    response = sr1(syn_packet, timeout=self.timeout, verbose=0)
    if response and response.haslayer(TCP):
        if response[TCP].flags == 0x12:  # SYN-ACK
            return True
    return False

Conclusion

PyPortScan provides a lightweight and flexible solution for identifying open ports and active hosts on a network. By supporting both standard and stealth scanning, it balances usability and stealth for different scenarios. The modular design also makes it straightforward to extend the tool with additional features such as banner grabbing, service identification, or exporting results in machine-readable formats.
